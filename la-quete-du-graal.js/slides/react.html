C'est une technique **interne**.
C'est comme si Angular nous avait séduit avec les zones...
Imaginez un vendeur pour une télévision avec comme 1er argument "TV génial : elle a le condensateur dernier cri type MKP 0,68 microfarad => 0,3 seconde + rapide à démarrer et vous économisez 10cts sur l'anée !"

Flux

Facebook Flux, Flummox, Alt, Fluxxor, Flux This, MartyJS, McFly, Fluxible, Delorean, Lux, Reflux, OmniscientJS, Fluxy, Material Flux, rx-flux, Redux (flux++)...

Les gens étaient un peu perdu avec react au début.
Facebook a présenté flux non pas sous la forme d'une lib mais juste une page web expliquant le truc.
Beaucoup d'implémentations sont apparues, certaines ont disparues.
Est-ce que c'est un petit exemple de ce que nous vivons à plus grande échelle avec l'histoire des frameworks ?

JS ALL THE THINGS
